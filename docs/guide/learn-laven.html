Primitive types

Laven is a dynamically typed language, and it provides a variety of primitive data types.

Scalar types

  • Numbers: 1, 3.14, 0.0
  • Strings: "hello, world", "a", "Yahallo!"
  • Bool: either true or false
  • Null: null

Compound types

  • Arrays: [], [2,3,5,7]
  • Maps:
Map{
    "name": "Kamille",
    "age": 29,
}

The compound types are stored as a reference unlike scalar types which are stored as a value

Example: (scalar types)

let num1 = 5;
let num2 = 21;

//Assign num1 to num2
num2 = num1;

//Assign num1 to 16
num1 = 16;

//Prints: num1 = 16 , num2 = 5
println("num1 =" , num1 , ", num2 =" , num2);
