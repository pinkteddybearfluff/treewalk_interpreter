Guide to using Laven Language

Basic Array usage

Arrays in Laven are dynamically sized. The elements in array are stored sequentially in memory.

Since Arrays in Laven are dynamically sized, it means that elements can be added to or removed from the Array.

Example code for basic usage with array

let arr = [2,3,5,7];

//Add 11 to the end of array
arr.push(11);
//Prints: [2, 3, 5, 7, 11]
println(arr);

//Remove element from the end of array
arr.pop();
//Prints: [2, 3, 5, 7]
println(arr);

//Get size of or number of elements in array, Prints: 4
println(arr.size());

//Remove element from the start of array
arr.popFront();
//Prints: [3, 5, 7]
println(arr);

Unlike primitive types which are stored by value, Arrays are stored as a reference. This might lead to unexpected behavior for some such a the behavior documented below.

Example: (compound types)

//Array [1, 2, 3, 4] is created and assigns reference of this Array to arr
//as if arr is alias to Array [1, 2, 3, 4]
let arr = [1, 2, 3, 4];

//Assign arr to list;
let list = arr;
//Now list and arr point/reference to same Array [1, 2, 3, 4];
//Assign first element of list to 100
list[0] = 100;
//Since list and arr reference the same Array [1, 2, 3, 4]
//changing first element of list also changes the first element of arr

//Prints: arr = [100, 2, 3, 4]
println("arr =", arr);
