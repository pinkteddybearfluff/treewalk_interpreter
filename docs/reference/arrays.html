Array.contains?()

Checks if the array contains the given element.


let arr = ["1", "5", "7", "9"];

println(arr.contains?("5"));
//Expected output: true

Parameters

searchElement

The value to search for in the array.

Return value

Returns true if searchElement exists in the array, else returns false.

Examples


[4, 1, 4, 5].includes(5);  //true
["hello", "konichiwa", "bye bye", "Ni Hao"].includes("welcome")  //false

Array.concat()

Joins 2 or more arrays to create new array without mutating the original ones. 


let arr = [1, 3, 5, 7];
println(arr.concat([9, 10, 14, 15], [16, 18]));
//Expected output: [1, 3, 5, 7, 9, 10, 14, 15, 16, 18]

Parameters

value1, ..., valueN

Returns a new Array with all the arrays and/or values merged.

If no arguments are provided then returns a new array which is shallow copy of the existing array.

Callout Note: Here shallow copy means that the array container is copied, which means that the primitive types are copied where as for the compound types and program defined types their references are copied not the referenced objects. Meaning changing elements in one array won't affect other array but changing the referenced object will show affect both arrays.

Return value

Returns a new Array.

Examples

Concatenating with arrays


let numbers = [1, 4, 7, 8];
let subArr1 = [5, 6, 7];
let subArr2 = [88, 99, 110];
let subArr3 = [0, 0, 1];

let newNumbers: Array[number] = numbers.concat(subArr1, subArr2, subArr3);

println(newNumbers);
// Expected output: [1, 4, 7, 8, 5, 6, 7, 88, 99, 110, 0, 0, 1]

Concatenating arrays with values


let greetings = ["hi", "hello", "yahallo", "yo"];
let subGreetings = ["hey", "wasup"];

let newGreetings: Array[str] = greetings.concat(subGreetings,"nihao");

println(newGreetings);
// Expected output: ["hi", "hello", "yahallo", "yo", "hey", "wasup", "nihao"]

Concatenating without arguments


let arr = [1, 2, 4, 6];
let b: Array[number] = arr.concat();  //assigns shallow copy of arr to b
b[0] = 100;

println(b, arr);
//Expected output: [100, 2, 4, 6] [1, 2, 4, 6]

Difference between regular assignment and Array.concat() copy


struct Point{
  x: number,
  y: number,

  fn stringify(self){
    return format("({}, {})", self.x, self.y);
  }
}

let arr = [Point(1, 2), Point(2, 2), Point(4,4)];

let shallowArr1 = arr;
//changing first element of shallowArr1 has also changed the arr
shallowArr1[0] = Point(0, 0);
println(format("arr: {}, shallow copied array: {}", arr, shallowArr1));
//Expected Output: arr: [(0, 0), (2, 2), (4, 4)], shallow copied array: [(0, 0), (2, 2), (4, 4)]

//Now arr is [(0, 0), (2, 2), (4, 4)]
let shallowArr2 = arr.concat();
//changing first element of shallowArr2 doesn't change arr
shallowArr2[0] = Point(10, 10);
//arr is still [(0, 0), (2, 2), (4, 4)]
println(format("arr: {}, shallow copied with concat: {}", arr, shallowArr2));
//Expected Output: arr: [(0, 0), (2, 2), (4, 4)], shallow copied with concat: [(10, 10), (2, 2), (4, 4)]

//changing the object the second element of shallowArr2 references also affects arr
shallowArr2[1].x = 100;
shallowArr2[1].y = 100;
//Now arr is also [(0, 0), (100, 100), (4, 4)]
println(format("arr: {}, shallow copied with concat: {}", arr, shallowArr2));
//Expected output: arr: [(0, 0), (100, 100), (4, 4)], shallow copied with concat: [(10, 10), (100, 100), (4, 4)]

In pointer terminology, arr is just a pointer to the actual Array object. And the Array stores pointers to the Point objects. Hence shallowArr1 = arr just means that shallowArr1 holds the same pointer and points to the same object. Therefore mutating the object pointed by shallowArr1 also affects arr.

But shallowArr2 = arr.concat() is the same as creating a new Array object and copying all the elements inside arr into shallowArr2. Hence shallowArr2 and arr point to different Array. But now the if the elements were just pointers then they are copied as is which means they point to the same object. Here the elements in both arrays point to the same Point objects. So mutating Array pointed by shallowArr2 won't mutate the Array pointed by arr, but if an element in both arrays points to the same object then mutating this object will affect both arrays.

Array.isEmpty?()

Checks if array is empty.

 
let arr = [2, 3];
println(arr.isEmpty?());
//Expected output: false

Parameters

None

Return value

Returns true if array is empty or Array.size() == 0 else returns false.

Example


[].isEmpty?();  //true
[""].isEmpty?();    //false
[null].isEmpty?();  //false

Array.map()

Returns a new array with a function applied on each element.

 
fn sqr(value) -> number{
    return value*value;
}

let arr = [1, 2, 3, 4];
let a = arr.map(sqr);
println(a);
//Expected output: [1, 4, 9, 16]

Parameters

function is applied on each element of the array and its return value is used in their place.

Return value

Returns a new array with element being the return value of function used on element of array.

Example


import math as m

let sqrs = [1,4,9,16];
println(sqrs.map(m.sqrt));
//Expected output: [1, 2, 3, 4]

let numbers = [3.14, 2.71, 6.04, 9.81];
println(numbers.map(m.floor));
//Expected output: [3, 2, 6, 9]

Array.slice()

Returns a shallow copy of portion of array from elements from start till end.

Syntax


          slice()
          slice(start)
          slice(start, end)

Parameters

start

Marks the start of the portion for the new array. The element at the start position is included in the new array. Can be negative. Is optional.

end

Marks the end of the portion for new array. The element at the end position is not included in the new array. Can be negative. Is option.

Return value

Returns a new shallow copied portion of the array including the element at start position till the end and not including the element at the end position.

Example

Slicing with start and end

println([1, 2, 3, 4, 5].slice(2,4));
//Expected output: [3, 4]
Slicing with negative positions

println([1, 2, 3, 4, 5, 6].slice(-6,-1));
//Expected output: [1, 2, 3, 4, 5]
Slicing with only start position

println([1, 2, 3, 4, 5, 6].slice(2));
//Expected output: [3, 4, 5, 6]
Slicing without arguments

//prints shallow copy of original
println([1,2,3,4,5].slice());
//Expected output: [1, 2, 3, 4, 5]

Array.sort()

Returns a new shallow copied array with its element sorted with the help of compareFn .

Syntax

sort(compareFn)

Parameters

compareFn is used to sort the array in desired order.

The compareFn takes two arguments a and b and returns a positive value when a comes after b, returns negative value when a comes before b and zero when both come at same place.

zero when both are considered equal.

Return value

Returns a sorted new shallow copied array.

Example


import math as m

struct Vector{
    x: number,
    y: number,

    fn magnitude(self){
        return m.sqrt(m.pow(self.x, 2) + m.pow(self.y,2));
    }

    fn stringify(self){
        return format("({}, {})", self.x, self.y);
    }
}

fn cmpFn(p1: Point, p2: Point) -> number{
    return p1.magnitude() - p2.magnitude();
}

let arr = [Vector(1,2), Vector(-2,3), Vector(0,1), Vector(4,5)];
//Sorts arr in order of Vectors with increasing magnitudes
println(arr.sort(cmpFn));
