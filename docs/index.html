Why Laven?

Simple

Simple syntax and easy to learn, provides basic important things out of the box and can get things done without hassle.

Expressive

With expressive syntax, laven is easy to read and understand, and offers support for procedural as well as functional programming style.

Safe

Laven is dynamically typed but has strict type checking, which means there are almost no errors that pass silently.

Laven is experimental interpreted programming language currently under active development. Although the language is now fairly usable, syntax, APIs, and standard library may change. Documentation is still under development.
